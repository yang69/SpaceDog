一些按键的说明

Caps Lock 键，短按是切换输入法，长按是大写锁定，大写锁定下短按默认回到中文输入法。

按住 Control 键的同时点击，等价于右击。

通用

  • Ctrl + A : Home
  • Command + ← ：Home
  • Command + → ：End
  • Fn + ↑ : 向上翻页
  • Fn + ↓ : 向下翻页
  • Fn + ← : 页面最开始
  • Fn + → : 页面最后
  • Command + A ：全选
  • Command + C ：复制
  • Command + V ：粘贴
  • Command + F ：查找
  • Command + S ：保存
  • Command + , ：设置

浏览器

  • 焦点移动到地址栏：Command + L
  • 新建标签页：Command + T
  • 关闭当前标签页：Command + W
  • 向右切换标签页：Ctrl + Tab
  • 向左切换标签页：Ctrl + Shift + Tab

finder

  • Command + Shift + . : 显示隐藏文件
  • Command + 方向键向上 : 返回父目录
  • Command + 方向键向下 : 进入子目录
  • Command + [ : 后退
  • Command + ] : 前进

系统

  • Command + Control + Q : 锁定系统
  • Command + Option + Esc : 强制退出应用程序
