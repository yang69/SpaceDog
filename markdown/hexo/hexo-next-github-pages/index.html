安装 hexo

初始化一个 hexo 项目

上述命令执行完，在当前目录下，会多一个“项目名”为名字的文件夹。

假设我们执行的是hexo init blog。进入blog文件夹，执行hexo s。
打开浏览器，输入“127.0.0.1:4000”，应该就可以看到一个 hello world 的界面了。
好了，回到刚才执行hexo s的命令行，按 Ctrl + C 结束执行。
